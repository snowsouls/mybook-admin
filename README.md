# mybook-admin

## describe

###### 这是一个后台管理系统，采用vue全家桶 + elementUl开发，后台动态控制路由展示。现主要功能为管理员、菜单、角色的增删改查，和日志展示。对应我的前端项目 https://github.com/snowsouls/mybook, 后续会继续添加功能。

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
